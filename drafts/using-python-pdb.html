Using python pdb

30 September 2017

To use Vim with pdb you need to have a few options enabled, to see the configuration :

:version

Personally I use Vim 8 built from source the build process is as follow :

git clone ...
cd ...

make distclean

./configure --with-python-config-dir=/usr/lib/python2.7/config-x86_64-linux-gnu \
--enable-gtk2-check \
--with-features=huge \
--enable-pythoninterp=yes

make
sudo make install

which vim

sudo pip install vimpdb vim .vimpdbrc

vim --serverlist python duplicate_in_string.py testeurdedebuger

PDB

To run a script to the debugger from the command line :

python -m pdb {script_name}.py

A few command to get by

s : step -> step to the next line
u : up      -> move up the stack
d : down    -> move down the stack

man pdb vim /usr/lib/python2.7/pdb.doc
